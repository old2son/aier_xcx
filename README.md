# 爱尔眼健康科普教育基地小程序

## 项目概述

这是一个基于 `uni-app + Vue 2` 开发的微信小程序项目，面向“爱尔眼健康科普教育基地”业务场景，提供首页展示、科普内容浏览、基地活动与预约、用户中心、积分体系、在线咨询、消息通知等功能。

项目当前以微信小程序为主要运行平台，使用分包方式组织业务页面，并结合 `uni-simple-router`、`Vuex` 与 Vant Weapp 组件完成页面导航、状态管理和交互体验。

## 核心功能

### 主页与内容展示

- 首页包含轮播图、预约入口、分类入口、精选推荐、基地动态等内容。
- 科普页提供分类导航、基地原创、热门推荐等科普内容浏览能力。
- 支持搜索、基地概况、参观须知、活动中心等内容型页面。

### 预约与活动

- 支持个人预约、团队预约、活动预约相关流程。
- 支持预约时段查询、剩余人数查询、我的预约、取消预约等能力。
- 支持常用观众管理，便于用户重复预约时快速填写信息。

### 用户与互动

- 支持手机号验证码登录、用户资料获取与编辑、头像上传。
- 支持收藏、浏览记录、积分、奖品兑换、签到等用户成长能力。
- 支持在线咨询、消息中心、意见反馈等互动模块。

### 登录与权限

- 路由层对预约、个人中心、积分、反馈、消息等页面做了登录校验。
- 请求封装支持按接口自动携带 `token`，并在登录失效时给出提示。

## 技术栈

- 框架：`uni-app`
- 前端：`Vue 2`
- 状态管理：`Vuex`
- 路由：`uni-simple-router`
- UI 组件：`@vant/weapp`、`uni-ui`
- 日期处理：`dayjs`
- 页面路由生成：`uni-read-pages`

## 页面结构

### 主 Tab 页面

- `首页`：`pages/tabBar/home/home`
- `科普`：`pages/tabBar/science/science`
- `导览`：`pages/tabBar/guide/guide`
- `我的`：`pages/tabBar/mine/mine`

说明：当前“导览”页面进入后会提示“暂未开放”。

### 分包页面

- `packageLogin`：登录、隐私政策
- `packageHomeDetail`：精选推荐、基地动态、搜索
- `packageAppointment`：预约入口、个人预约、团队预约
- `packageCategory`：活动中心、基地概况、参观须知、科普分类内容
- `packageMine`：个人资料、我的预约、常用观众、积分、消息、反馈、在线咨询等

## 接口与运行说明

### 接口服务

- 当前接口基础地址为：`https://api.grandeffect.cn`

### 运行环境

- 当前项目配置为 `Vue 2` 的 `uni-app` 工程。
- `manifest.json` 中已配置微信小程序 `appid`。
- 项目未在 `package.json` 中提供完整的本地开发脚本，通常适合使用 `HBuilderX` 导入后运行到微信开发者工具。

### 依赖安装

```bash
npm install
```

如使用 `HBuilderX`：

1. 导入项目根目录。
2. 运行到微信开发者工具。
3. 在微信开发者工具中完成预览、调试与上传。

## 项目特点

- 使用分包降低主包体积，适合小程序业务拆分。
- 首页与科普页采用较多内容化展示组件，偏运营展示型应用。
- 用户中心、预约、积分、咨询等模块完整，业务闭环较清晰。
- 全局引入 Vant Weapp 组件，便于快速搭建表单、弹窗、选择器等交互。

### 预约限制
- 人数 <= 40
- 时间 <= 1小时
